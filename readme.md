# Back end Projeto e-Commerce Dio
Esta é o repositório de back end do projeto final do bootcamp da Inter pela plataforma [DIO](https://digitalinnovation.one/)

## Tech Stack
* Node.js
* Typescript
* Express.js
* SQL-lite
  

## Como iniciar

npm run dev
## Dependencias 
* express js 
* cors

## Endpoints

* GET /message
* POST /message/
